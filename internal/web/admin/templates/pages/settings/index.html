Exit Agent — 持仓中决策 worker ?作用:scorer 只在开仓时打分,持仓中之后到平仓期间,bot 完全靠固定 SL/TP 守。Exit Agent 填这个空白 — 每 N 分钟扫一次所有 status='open' 的 virtual_position,把当前价 / 持仓时长 / K 线 / 宏观状态 / 历史胜率 / 钉选反思打包给 LLM,让它决定下一步动作。

四档动作:
• 继续拿 (hold):继续持有
• 上移止损 (tighten_sl):把止损往上挪锁定盈利
• 部分止盈 (take_partial):平 ≤50% 锁部分利润,剩余继续跑
• 立即平仓 (exit_now):全平止损

所有决策落 agent_exit_decisions 表,可在 /eval/exit 看,每条决策 60min 后自动回填"如果当时 hold 会怎样"作为对照。

⚠️ 切到 active 之前请在 shadow 至少观察 1-2 周再开,先确认 LLM 决策合理。

📖 怎么拿 webhook URL(点击展开)
  1. 电脑端打开企业微信,进入要接收消息的群
  2. 右上角"…" → "群机器人" → "添加机器人"
  3. 取个名字(如 "tvbot-news"),头像可选 → "添加"
  4. 添加成功后会显示 Webhook 地址,形如 https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=XXX
  5. 复制整条地址粘贴到下方"Webhook URL"输入框

⚠ 这个 URL 等于群发权限,不要泄漏。
