企业微信群机器人通知

把 News 评估结果推送到企业微信群。News worker 每次跑完会判断 impact 是否达到阈值，达到才推送。

📖 怎么拿 webhook URL（点击展开）
  1. 电脑端打开企业微信，进入要接收消息的群
  2. 右上角"…" → "群机器人" → "添加机器人"
  3. 取个名字（如 "tvbot-news"），头像可选 → "添加"
  4. 添加成功后会显示 Webhook 地址，形如 https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=XXX
  5. 复制整条地址粘贴到下方"Webhook URL"输入框

⚠ 这个 URL 等于群发权限，**不要泄漏**。

默认 medium，避免群里被低质消息刷屏。
