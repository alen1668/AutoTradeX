Perp metrics — 历史快照

📖 这个页面是干什么用的

永续指标 worker 每 5 分钟从币安拉永续合约的三类市场情绪数据，按交易对入库。 这些数据会注入 scorer 的 prompt，让 LLM 在给 TradingView 信号打分时多看一层永续合约市场状态。 纯参考信息，不改 scorer 阈值，也不影响硬风控。

三个指标怎么读：

资金费率 多头/空头哪边在为持仓付钱。正值=多头付钱给空头（多头拥挤），极端正值常被反向 fade。
持仓量变化 跟价格一起读才有意义：价↑+持仓↑=新多入场（趋势强）；价↑+持仓↓=空头挤仓（多头脆）；价↓+持仓↑=新空入场；价↓+持仓↓=多头投降。
大户多空比 币安头部持仓账户的多空仓位比例。>1 偏多、<1 偏空，极端值代表大户集中下注。
分类标签的阈值（英文是 prompt 里实际用的 label，鼠标悬停每个 chip 可看）
  • 拥挤度：>0.05% 多头极度拥挤(extreme_long) / >0.01% 多头偏拥挤(mild_long) / ±0.01% 中性(neutral) / <-0.01% 空头偏拥挤(mild_short) / <-0.05% 空头极度拥挤(extreme_short)
  • 资金动向：需 |持仓 24h| ≥ 3% 且 |价格 24h| > 0.5% 才有方向，否则归为"无明显方向(neutral)"
  • 大户倾向：>2.0 大户强多 / >1.3 大户偏多 / 0.7~1.3 多空均衡 / 0.5~0.7 大户偏空 / <0.5 大户强空

worker 抓取的交易对 = 当前所有启用策略的交易对 ∪ BTCUSDT。数据库表 perp_metrics。
