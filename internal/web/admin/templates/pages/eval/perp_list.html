Perp metrics — 历史快照

📖 这个页面是干什么用的

Perp metrics worker 每 5 分钟从 binance 拉永续合约的三组指标 —— funding rate（拥挤方向）、open interest 24h 变化（资金进出 + 价格对照）、 top-trader long/short ratio（大户多空仓位比），按 symbol 入库。

这些指标会通过 macrocontext 注入 scorer 的 prompt 第 6 段（"永续指标"）， 让 LLM 在打分时多看一层永续合约市场情绪。纯增量信息，不动 scorer 阈值，也不动硬风控。

怎么读 derived labels：

  • funding：>0.05% extreme_long / >0.01% mild_long / ±0.01% neutral / <-0.01% mild_short / <-0.05% extreme_short
  • OI signal：要求 |OI 24h| ≥ 3% 且 |price 24h| > 0.5%。price↑+OI↑=new_longs（强趋势）/ price↑+OI↓=short_squeeze（空头补仓，多头脆）/ price↓+OI↑=new_shorts / price↓+OI↓=capitulation
  • top L/S：>2.0 strongly_bullish / >1.3 bullish / 0.7~1.3 balanced / 0.5~0.7 bearish / <0.5 strongly_bearish

worker 抓取的 symbol 列表 = 当前所有 enabled strategy 的 symbol ∪ BTCUSDT。 数据库表 perp_metrics；想直接 SQL 查也行。
